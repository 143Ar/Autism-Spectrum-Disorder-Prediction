# 🧠 Autism Spectrum Disorder (ASD) Prediction System

## 📌 Overview
This project focuses on early prediction of Autism Spectrum Disorder (ASD) using machine learning techniques. 
The system analyzes behavioral and screening data to estimate the likelihood of ASD, helping in early detection and awareness.

## 🚀 Features
- Predicts likelihood of ASD using machine learning model  
- Probability-based output for better interpretability  
- User-friendly web interface built with Streamlit  
- Data visualization for understanding patterns and insights  

## 🛠️ Tech Stack
- Python  
- Scikit-learn  
- Pandas, NumPy  
- Matplotlib, Seaborn  
- Streamlit  

## 📊 Workflow
1. Data Collection & Cleaning  
2. Exploratory Data Analysis (EDA)  
3. Feature Selection  
4. Model Building (Logistic Regression)  
5. Model Evaluation  
6. Deployment using Streamlit  

## 📈 Model Details
- Algorithm: Logistic Regression  
- Evaluation Metrics: Accuracy, Precision, Recall  
- Output: Probability score indicating ASD likelihood
